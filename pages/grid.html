Grid Example 25: grid-column & grid-row

Check the code and see the resulting layout below.

HTML Code:


          <div class="grid_container"> <-----Parent div.
            <div id="div_1">1</div> <---------child div.
            <div id="div_2">2</div>
            <div>3</div>
            <div>4</div>
            <div>5</div>
            <div>6</div>
          </div>

CSS Code:


            .grid_container {
              margin: 10px auto;
              padding: 3px;
              border: 2px solid darkgreen;
              width: 60%;
              height: 300px;
              display: grid;
              grid-template-columns: auto auto auto;
              grid-gap: 10px;
            }
            .grid_container div {
              background-color: palegreen;
              border: 2px solid darkgreen;
              border-radius: 5px;
              font-size: 1.5em;
              text-align: center;
            }
            #div_1 { grid-column: 1 / 4; }
            #div_2 { grid-row: 2 / 4; }

            <div id="area">2</div> <------------child div.

            <div>5</div>
            <div>6</div>
            <div>7</div>
            <div>8</div>

          </div>

CSS Code:


            .grid_container_2 {
              margin: 10px auto;
              padding: 3px;
              border: 2px solid darkgreen;
              width: 60%;
              height: 300px;
              display: grid;
              grid-template-columns: auto auto auto auto;
              grid-gap: 10px;
            }
            .grid_container_2 div {
              background-color: palegreen;
              border: 2px solid darkgreen;
              border-radius: 5px;
              font-size: 1.5em;
              text-align: center;
            }
            #area { grid-area: 1 / 2 / 3 / 4; }

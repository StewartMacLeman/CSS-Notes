Starting HTML and CCS

Below is the original HTML and CSS for the sample content.

HTML Code:


          <div> <----------Parent div.
            <div></div> <---child div.
          </div>

CSS Code:


          div { <----------Parent div.
            margin: 10px auto;
            border: 2px solid darkgreen;
            width: 60%;
            height: 300px;
            overflow-x: scroll;
          }

          div div { <-------child div.
            margin: 0px;
            width: 33.33%;
            height: 96px;
            background-color: palegreen;
            border: 2px solid darkgreen;
            border-radius: 5px;
          }

Starting HTML and CCS

Below is the original HTML and CSS for the sample content.

HTML Code:


          <div> <----------Parent div.
            <div>div</div> <---child div.
          </div>

CSS Code:


          div { <----------Parent div.
            margin: 10px auto;
            border: 2px solid darkgreen;
            width: 60%;
            height: 150px;
            display: flex;
            justify-content: center;
            align-items: center;
          }

          div div { <-------child div.
            background-color: palegreen;
            border: 2px solid darkgreen;
            width: 100px;
            height: 50px;
            border-radius: 5px;
            font-size: 1.5em;
            text-align: center;
          }

Animation Example 1: animation-name & animation-duration

Check the code below and click the button to see the change to the sample content.

CSS Code:


            @keyframes colour_change {
              from {background-color: linen;}
              to   {background-color: darkgreen;}
            }

            div {
              animation-name: colour_change;
              animation-duration: 5s
            }

Animation Example 7: animation-iteration-count

Check the code below and click the button to see the change to the sample content.

CSS Code:


            @keyframes box_move {
              0%   { top: 0px;  left: 0px; }
              25%  { top: 0px;  left: 250px; }
              50%  { top: 45px; left: 250px; }
              75%  { top: 45px; left: 0px; }
              100% { top: 0px;  left: 0px; }
            }

            div {
              animation-name: box_move;
              animation-duration: 5s;
              position: relative;
              animation-iteration-count: 2;
            }

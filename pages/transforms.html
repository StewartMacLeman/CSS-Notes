Starting HTML and CCS

Below is the original HTML and CSS for the sample content.

HTML Code:


          <div> <----------Parent div.
            <div>div</div> <---child div.
          </div>

CSS Code:


          div { <----------Parent div.
            margin: 10px auto;
            border: 2px solid darkgreen;
            width: 60%;
            height: 150px;
            display: flex;
            justify-content: center;
            align-items: center;
          }

          div div { <-------child div.
            background-color: palegreen;
            border: 2px solid darkgreen;
            width: 100px;
            height: 50px;
            border-radius: 5px;
            font-size: 1.5em;
            text-align: center;
          }

CSS Code:


          div { <----------Parent div.
            margin: 10px auto;
            border: 2px solid darkgreen;
            width: 60%;
            height: 100px;
          }

          div div { <-------child div.
            position: relative;
            top: 8%;
            margin: 0px auto;
            width: 40%;
            height: 80%;
            border: 2px solid darkgreen;
            border-radius: 5px;
          }

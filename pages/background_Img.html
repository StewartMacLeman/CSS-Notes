Starting HTML and CCS

Below is the original HTML and CSS for the sample content.

HTML Code:


          <div> <----------Parent div.
            <div></div> <---child div.
          </div>

CSS Code:


          div { <----------Parent div.
            margin: 10px auto;
            border: 2px solid darkgreen;
            width: 60%;
            height: 300px;
          }

          div div { <-------child div.
            position: relative;
            top: 9%;
            margin: 0px auto;
            width: 60%;
            height: 80%;
            border: 3px solid darkgreen;
          }
